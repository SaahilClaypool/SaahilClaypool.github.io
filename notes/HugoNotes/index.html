Hugo Notes

Hugo Reference

Hugo is a static HTML Site generator. It takes directories of markdown and compiles them into linked HTML websites.

Setup

  1. Install Hugo

    • a) Install the binary from apt

      $ sudo apt install hugo

    • b) Install through go

      $ go get -u -v github.com/spf13/hugo

Commands

  1. Create new content

     $ hugo new <path>/<thing>.md

  2. Serve Site Locally

    $ hugo server

Deploying to Github

Using this link: http://gohugo.io/tutorials/github-pages-blog/

$ ./deploy.sh

NOTE To get a theme, do git submodule add NOTE Must make a ssh key for the computer that is pushing to github.
